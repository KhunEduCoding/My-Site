មេរៀនទី ១: ការណែនាំពីកម្មវិធី

  • គោលបំណង

  • បន្ទាប់ពីបញ្ចប់មេរៀននេះអ្នកអាច៖

    • ស្គាល់ពីអត្ថប្រយោជន៍នៃការប្រើប្រាស់ Toolbars

    • ចេះរៀបចំ Page Setup

    • ចេះប្រើប្រាស់ Row និង Column

    • ចេះប្រើប្រាស់ Workbooks និង Worksheet

    • ចេះប្រើប្រាស់ Tips និង Trick ជាដើម។

  • និយមន័យ

  • Excel ឬ Microsoft Excel គឺជាឈ្មោះដែលកម្មវិធីដែលបង្កើតឡើងដោយក្រុមហ៊ុន Microsoft ត្រូវបានគេស្គាល់ ប្រើប្រាស់យ៉ាងទូលំទូលាយដោយក្រុមហ៊ុន និងបុគ្គលដើម្បីអនុវត្តប្រតិបត្តិការហិរញ្ញវត្ថុ និងគណនេយ្យដោយប្រើសៀវភៅបញ្ជី។ Microsoft Excel ប្រើសម្រាប់ការគណនាសាមញ្ញ និងស្មុគស្មាញ បង្កើតបញ្ជីទិន្នន័យ បង្កើតរបាយការណ៍ និងក្រាហ្វដ៏ស្មុគ្រស្មាញ ការព្យាករណ៍ និងការវិភាគនិន្នាការ ការវិភាគស្ថិតិ និងហិរញ្ញវត្ថុ បន្ថែមពីលើការមានភាសាកម្មវិធីរួមបញ្ចូលគ្នាដែលមានមូលដ្ឋាននៅលើ Visual Basic ។

  • ការបើកកម្មវិធី

  • របៀបនីមួយៗ៖

    • របៀបទី១៖ ចុច Double Click លើ Icon (Shortcut) ដែលមាននៅលើ Desktop ស្រាប់

    • របៀបទី២៖ ចុច "Windows" => វាយពាក្យ "Excel" => Open

    • របៀបទី៣៖ ចុច "Windows + R" => វាយពាក្យ "Excel" => Enter

  • ការបង្កើតឯកសារថ្មី

    • របៀបទី១៖ ចុចជ្រើសរើសយក Blank Workbook បន្ទាប់ពីដំណើរការបើកកម្មវីធី

    • របៀបទី២៖ ចុច "File" => "New" => Blank Workbook

    • របៀបទី៣៖ ចុច "Ctrl + N"

  • ចាប់ផ្តើមកម្មវិធី

    • រូបភាពផ្ទាំងកម្មវិធី Microsoft Excel

    • នៅក្នុង Workbook នៃកម្មវិធីនមានសន្លឹកកិច្ចការ (Worksheet) ជាច្រើន ហើយនៅក្នុងសន្លឹកកិច្ចការនីមួយៗមានជួរដេក (Row) ចំនួន 1048576 និងជួរឈរ (Column) ចំនួន 16384។

    • ➤ Tab Bar គឺជាកន្លែងមួយដែលមានផ្ទុក Tools ជាច្រើនសម្រាប់អនុវត្តន៍ការងារ

    • ➤ Active Cell គឺជាកន្លែងសម្រាប់បង្ហាញអាសយដ្ឋានរបស់ Cell

    • ➤ Formula Bar គឺជាកន្លែងសម្រាប់បង្ហាញរូបមន្ត ឬទិន្នន័យក្នុង Cell នីមួយៗ

    • ➤ Row គឺជាជួរដេកនៃ Cell

    • ➤ Columns គឺជាជួរឈរនៃ Cell

    • ➤ Cell គឺជាចំណុចប្រសព្វរវាង Row និង Columns

    • ➤ Work Book គឺជាស្រទាប់ការងាររបស់កម្មវិធី ហើយវាគ្រប់គ្រងលើសន្លឹកកិច្ចការ

    • ➤ Work Sheet គឺស្ថិតនៅក្រោមសៀវភៅកិច្ចការ (Work Book) ដែលគ្រប់គ្រងលើ Cell។

មេរៀនទី ២: មុខងារមូលដ្ឋាន

សិក្សាអំពីមុខងារមូលដ្ឋានដូចជា SUM, AVERAGE, MIN, និង MAX ។

មេរៀនទី ៣: រូបមន្ត

យល់ពីវិធីបង្កើតនិងប្រើរូបមន្តក្នុង Excel សម្រាប់ការគណនា។

មេរៀនទី ៤: ការបង្ហាញទិន្នន័យ

សិក្សាវិធីបង្កើតគំនូសតាងនិងគំនូសបង្ហាញទិន្នន័យ។

មេរៀនទី ៥: តារាង Pivot

សិក្សាអំពីការប្រើតារាង Pivot សម្រាប់ការសង្ខេបនិងវិភាគទិន្នន័យ។

មេរៀនទី ៦: មុខងារខ្ពស់

ស្វែងយល់អំពីមុខងារខ្ពស់ដូចជា VLOOKUP, HLOOKUP, និង IF statements។

មេរៀនទី ៧: វិភាគទិន្នន័យ

បច្ចេកទេសសម្រាប់វិភាគទិន្នន័យក្នុង Excel រួមមានការត្រងនិងការរៀបចំ។

មេរៀនទី ៨: Macros

ការណែនាំអំពីការបង្កើតនិងប្រើ Macros សម្រាប់ស្វ័យប្រវត្តិកម្មការងារនៅក្នុង Excel។

មេរៀនទី ៩: ការធ្វើការរួមគ្នា

សិក្សាវិធីធ្វើការរួមគ្នាជាមួយអ្នកដទៃដោយប្រើលក្ខណៈពិសេសនៃការចែករំលែកនិងការរួមការងាររបស់ Excel។
